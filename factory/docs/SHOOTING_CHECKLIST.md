# Shooting checklist (the highest-ROI file in this repo)

No pipeline fixes weak raw footage. Ten minutes of shooting discipline beats an
hour of revision loops. Tape this next to the camera.

## Before recording
- [ ] Frame yourself center-ish with headroom — the 9:16 crop takes the middle
      ~56% of a 16:9 frame. Anything at the edges is gone.
- [ ] Props (steel plate, hardware wallet, paper) within that center band, chest height.
- [ ] Audio check: quiet room, mic close. Whisper is good, not magic.

## While recording
- [ ] Record 3 different HOOK takes for the same idea. The selector can only pick
      from hooks that exist. ("This is how people lose Bitcoin forever" / the price
      contrast / the demonstration open.)
- [ ] Speak in self-contained 15–40s blocks: hook → point → concrete payoff → stop.
      Never reference "earlier" or "as I said."
- [ ] Botched a line? Pause 2 full seconds, restart the sentence. The 2s gap is a
      clean cut point and the tightener keeps the best take.
- [ ] Say numbers explicitly ("twenty-one dollars, not a hundred twenty") — numbers
      make the best pop-ups and the selector scores concrete payoffs higher.
- [ ] Physically point/hold the product when relevant — arrows and circles need a
      target on screen.

## After recording
- [ ] One idea per project run. A 10-minute ramble across 4 topics gives the
      selector mush; 3 minutes on one topic gives it gold.
